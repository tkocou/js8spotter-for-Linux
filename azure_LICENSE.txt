# JS8Spotter de KF7MIX
#
# A JS8Call API mechanism to keep track of activity containing specific search
# terms, including callsigns or other activity. Matches on RX.ACTIVITY,
# RX.DIRECTED, and RX.SPOT only. Tested with JS8Call v2.2.0
#
# Enable TCP API in JS8Call. File>Settings>Reporting, checkmark on Allow setting
# station information, Enable TCP Server API, Accept TCP Requests.
